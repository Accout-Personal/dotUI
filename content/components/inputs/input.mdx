## Usage

```jsx
import { Input } from "@/components/core/input";
```

```jsx
<Input placeholder="email" />
```

## API Reference

| Prop             | Type                   | Default |
| ---------------- | ---------------------- | ------- |
| `size`           | `"sm" \| "md" \| "lg"` | `md`    |
| `disabled`       | `boolean`              | -       |
| `prefix`         | `React.ReactNode`      | -       |
| `suffix`         | `React.ReactNode`      | -       |
| `loading`        | `boolean`              | -       |
| `loaderPosition` | `"prefix" \| "suffix"` | -       |
| `status`         | `"error" \| "success"` | -       |
