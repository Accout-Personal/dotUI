## Usage

```jsx
import { InputOTP } from "@/components/ui/input-otp";
```

```jsx
<InputOTP maxLength={6} />
```

## API Reference

### InputOTP

| Prop         | Type                                | Default     |
| ------------ | ----------------------------------- | ----------- |
| `length*`    | `number`                            | -           |
| `value`      | `number`                            | -           |
| `onChange`   | `func: (value: string) => void`     | -           |
| `onComplete` | `func: (value: string) => void`     | -           |
| `size`       | `"sm" \| "md" \| "lg"`              | `"md"`      |
| `variant`    | `"default" \| "success" \| "error"` | `"default"` |
| `disabled`   | `boolean`                           | `"default"` |

### InputOTPRoot

| Prop         | Type                            | Default     |
| ------------ | ------------------------------- | ----------- |
| `length`     | `number`                        | -           |
| `value`      | `number`                        | -           |
| `onChange`   | `func: (value: string) => void` | -           |
| `onComplete` | `func: (value: string) => void` | -           |
| `disabled`   | `boolean`                       | `"default"` |

### InputOTPSlot

| Prop       | Type                                | Default     |
| ---------- | ----------------------------------- | ----------- |
| `length`   | `number`                            | -           |
| `size`     | `"sm" \| "md" \| "lg"`              | `"md"`      |
| `variant`  | `"default" \| "success" \| "error"` | `"default"` |
| `disabled` | `boolean`                           | `"default"` |
