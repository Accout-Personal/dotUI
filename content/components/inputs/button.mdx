## API Reference

| Prop      | Type                                                                                             | Default   |
| --------- | ------------------------------------------------------------------------------------------------ | --------- |
| `asChild` | `boolean`                                                                                        | -         |
| `variant` | `"primary" \| "neutral" \| "link" \| "ghost" \| "outline" \| "danger" \| "success" \| "warning"` | `neutral` |
| `size`    | `"sm" \| "md" \| "lg"`                                                                           | `md`      |
| `shape`   | `"default" \| "square" \| "circle"`                                                              | `default` |
| `loading` | `boolean`                                                                                        | -         |
| `prefix`  | `React.ReactNode`                                                                                | -         |
| `suffix`  | `React.ReactNode`                                                                                | -         |
