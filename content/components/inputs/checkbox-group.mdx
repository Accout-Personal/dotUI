## Accessibility

### Keyboard interactions

| Key         | Description                                        |
| ----------- | -------------------------------------------------- |
| `Tab`       | Moves focus to the next checkbox in the group.     |
| `Shift+Tab` | Moves focus to the previous checkbox in the group. |
| `Space`     | Checks/unchecks the focused checkbox.              |
