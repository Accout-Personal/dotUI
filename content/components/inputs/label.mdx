```bash
npm install @radix-ui/react-label
```

  <Step>Copy and paste the following code into your project.</Step>
  <ComponentSource name="components/core/default/label" />
  <Step>Update the import paths to match your project setup.</Step>

## Usage

```jsx
import { Label } from "@/components/core/label";
```

```jsx
<Label htmlFor="email">Email address</Label>
```

## API Reference

| Prop                                                                                                                       | Type      | Default |
| -------------------------------------------------------------------------------------------------------------------------- | --------- | ------- |
| `asChild`<Info>Change the default rendered element for the one passed as a child, merging their props and behavior.</Info> | `boolean` | -       |
| `htmlFor`<Info>The id of the element the label is associated with.</Info>                                                  | `string`  | -       |
