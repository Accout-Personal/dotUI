## Usage

Use a text input to allow users to input custom text entries with a keyboard.

## Options

### Variants

Use the `variant` prop to control the visual style of the TextField.<br/>
The default variant is `default`.

<ComponentPreview name="demos/components/core/text-field/variants" className="max-w-[250px]" />

### Sizes

Use the `size` prop to control the size of the TextField.<br/>
The default variant is `md`.

### Label

A visual label should be provided for the TextField using the label prop or a hidden label using `aria-label` prop.<br/>
If the TextField is required, the isRequired and necessityIndicator props can be used to show a required state.

<ComponentPreview name="demos/components/core/text-field/label" className="max-w-[250px]" />

### Description

A description can be supplied to a TextField via the `description` prop. The description is always visible unless the `validationState` is `“invalid”` and an error message is provided.
