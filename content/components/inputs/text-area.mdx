## Usage

Use a textarea to allow users to input a long string of free-form text.

- Always include a label (it can be visible or hidden with `aria-label`)

## Options

### Variants

Use the `variant` prop to control the visual style of the `TextArea`.<br/>
The default variant is `default`.

### Label

A visual label can be provided for the `TextArea` using the label prop or a hidden label using `aria-label` prop.

<ComponentPreview name="demos/components/core/text-area/label" className="max-w-[250px]" />

### Description

A description can be supplied to a TextArea via the `description` prop. The description is always visible unless the `validationState` is `“invalid”` and an error message is provided.

<ComponentPreview name="demos/components/core/text-area/description" className="max-w-[250px]" />

A ContextualHelp element may be placed next to the label to provide additional information or help about a TextArea.

### Error message

An error message can be supplied to a TextArea, which will be displayed when the `validationState` is `“invalid”`.

<ComponentPreview name="demos/components/core/text-area/error-message" className="max-w-[250px]" />

### Loading

Use the `isLoading` prop to control the loading state of the TextArea.
Use the `loaderPosition` prop to control the position of the loader.

<ComponentPreview name="demos/components/core/text-area/loading" className="max-w-[250px]" />

### Disabled

Use the `isDisabled` prop to disable the TextArea.

<ComponentPreview name="demos/components/core/text-area/disabled" className="max-w-[250px]" />

### ReadOnly

The isReadOnly boolean prop makes the TextArea's text content immutable. Unlike isDisabled, the TextArea remains focusable and the contents can still be copied.

<ComponentPreview name="demos/components/core/text-area/read-only" className="max-w-[250px]" />

### Required

Use the `isRequired` prop to mark the TextArea as required.
Use the `necessityIndicator` prop to control the visual style of the required state.

## API Reference

### TextArea

This component inherits and merges props from TextFieldRoot and Field parts.

| Prop             | Type                                                           | Default     | Description                                 |
| ---------------- | -------------------------------------------------------------- | ----------- | ------------------------------------------- |
| `variant`        | `"default" \| "primary" \| "danger" \| "success" \| "warning"` | `"default"` | The visual style of the TextArea.           |
| `size`           | `"sm" \| "md" \| "lg"`                                         | `"md"`      | The size of the input.                      |
| `prefix`         | `React.ReactNode`                                              | -           | A visual to display before the input.       |
| `suffix`         | `React.ReactNode`                                              | -           | A visual to display after the input.        |
| `isLoading`      | `boolean`                                                      | -           | Whether the TextArea is in a loading state. |
| `loaderPosition` | `"prefix" \| "suffix"`                                         | -           | The loader position                         |
