## Installation

<Steps>
  <Step>Install the following dependencies:</Step>

```bash
npm install sonner
```

{" "}

<Step>Copy and paste the following code into your project.</Step>
<ComponentSource name="components/core/default/toast" />
<Step>Update the import paths to match your project setup.</Step>
<Step>Add the Toaster component to your root layout</Step>

```tsx
import { Toaster } from "@/components/core/toast";

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>
        {children}
        <Toaster />
      </body>
    </html>
  );
}
```

</Steps>

## Usage

```jsx
import { toast } from "@/components/core/toast";
```

```jsx
<Button onClick={() => toast("My first toast")}>Give me a toast</Button>
```

## API Reference

### toast

These are the available options for the `toast` function, such as `toast.success`, `toast.error`, `toast.warning` and `toast.info`.

#### Params

| Params     | Type                        | Default |
| ---------- | --------------------------- | ------- |
| `message*` | `string \| React.ReactNode` | -       |
| `options`  | `ToastOptions`              | -       |

#### ToastOptions

| Prop                | Type                                                                                                                                                                        | Default          |
| ------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------- |
| `description`       | `string \| React.ReactNode`                                                                                                                                                 | -                |
| `variant`           | `"default" \| "muted" \| "fill"`                                                                                                                                            | `"default"`      |
| `important`         | `boolean`                                                                                                                                                                   | `false`          |
| `duration`          | `number`                                                                                                                                                                    | `4000`           |
| `position`          | `Position` <Info className="min-w-none"><Code>"top-left" \| "top-right" \| "top-center" \| "bottom-left" \| "bottom-right" \| "bottom-center"</Code></Info>                 | `"bottom-right"` |
| `dismissible`       | `boolean`                                                                                                                                                                   | `true`           |
| `icon`              | `React.ReactNode`                                                                                                                                                           | -                |
| `id`                | `string \| number`                                                                                                                                                          | -                |
| `action`            | `{ label: string; onClick: function }` <Info className="min-w-none"><Code>\{ label: string; onClick: (event: React.MouseEvent\<HTMLButtonElement\>) => void }</Code></Info> | -                |
| `onAutoClose`       | `() => void`                                                                                                                                                                | -                |
| `onUndo`            | `() => void`                                                                                                                                                                | -                |
| `showDismissButton` | `boolean`                                                                                                                                                                   | `true`           |
