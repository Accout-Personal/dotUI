## Usage

Use a `DatePicker` to allow users to enter and edit time values.

## Options

### Label

A visual label can be provided for the `DatePicker` using the label prop or a hidden label using `aria-label` prop.<br/>

<ComponentPreview name="demos/components/core/date-picker/label" />

### Sizes

Use the `size` prop to control the size of the `DatePicker`.<br/>
The default variant is `md`.

<ComponentPreview name="demos/components/core/date-picker/sizes" />

### Prefix

To add additional context for the `DatePicker`, use the `prefix`prop.

<ComponentPreview name="demos/components/core/date-picker/prefix" />

### Description

A description can be supplied to a DatePicker via the `description` prop. The description is always visible unless the `validationState` is `“invalid”` and an error message is provided.

<ComponentPreview name="demos/components/core/date-picker/description" />

### Contextual help

A ContextualHelp element may be placed next to the label to provide additional information or help about a DatePicker.

<ComponentPreview name="demos/components/core/date-picker/contextual-help" />

### Error message

An error message can be supplied to a DatePicker, which will be displayed when the `validationState` is `“invalid”`.

<ComponentPreview name="demos/components/core/date-picker/error-message" />

### Time zones

DatePicker is time zone aware when a ZonedDateTime object is provided as the value.

<ComponentPreview name="demos/components/core/date-picker/time-zones" />

### Granularity

The `granularity` prop allows you to control the smallest unit that is displayed by DatePicker.

<ComponentPreview name="demos/components/core/date-picker/granularity" />

### Placeholder value

Use the `placeholderValue` prop to control the default values of each segment. The default value is midnight.

<ComponentPreview name="demos/components/core/date-picker/placeholder" />

### Hide time zone

Use the `hideTimeZone` prop to hide the time zone abbreviation.

<ComponentPreview name="demos/components/core/date-picker/hide-time-zone" />

### Hour cycle

Use the `hourCycle` prop to control the hour cycle of the DatePicker.

<ComponentPreview name="demos/components/core/date-picker/hour-cycle" />

### Loading

Use the `isLoading` prop to control the loading state of the DatePicker.

<ComponentPreview name="demos/components/core/date-picker/loading" className="max-w-xs" />

### Disabled

Use the `isDisabled` prop to disable the DatePicker.

<ComponentPreview
  name="demos/components/core/date-picker/disabled"
  className="max-w-[250px]"
/>

### ReadOnly

The isReadOnly boolean prop makes the DatePicker's text content immutable. Unlike isDisabled, the DatePicker remains focusable and the contents can still be copied.

<ComponentPreview
  name="demos/components/core/date-picker/read-only"
  className="max-w-[250px]"
/>

### Required

Use the `isRequired` prop to mark the DatePicker as required.
Use the `necessityIndicator` prop to control the visual style of the required state.

<ComponentPreview
  name="demos/components/core/date-picker/required"
  className="max-w-sm"
/>

## Accessibility

### Keyboard interactions

| Key          | Description                          |
| ------------ | ------------------------------------ |
| `Tab`        | Moves focus to the first segment.    |
| `ArrowRight` | Moves focus to the next segment.     |
| `ArrowLeft`  | Moves focus to the previous segment. |
| `ArrowUp`    | Increments the value of the segment. |
| `ArrowDown`  | Decrements the value of the segment. |
