## Usage

ToggleButtons allow users to toggle a selection on or off, for example, switching between two states or modes.<br />
They can have a label, an icon, or both. An icon is provided by passing an icon component as child to the ToggleButton or by using the `prefix` and `suffix` props. A visible label can be provided by passing a string as a child.

## Options

### Variants

Use the `variant` prop to control the visual style of the toggle button.<br/>
The default variant is `quiet`.

<ComponentPreview name="demos/components/core/toggle-button/variants" />

### Sizes

Use the `size` prop to control the size of the toggle button.<br/>
The default variant is `md`.

<ComponentPreview name="demos/components/core/toggle-button/sizes" />

### Shapes

Use the `shape` prop to control the shape of the toggle button.<br/>
The default variant is `rectangle`.<br/>
Icon-only toggle buttons should include an `aria-label`.

## API Reference

| Prop      | Type                                  | Default    |
| --------- | ------------------------------------- | ---------- |
| `variant` | `"quiet" \| "outline" \| "accent"`    | `"quiet"`  |
| `size`    | `"sm" \| "md" \| "lg"`                | `"md"`     |
| `shape`   | `"rectangle" \| "square" \| "circle"` | `"square"` |
| `prefix`  | `React.ReactNode`                     | -          |
| `suffix`  | `React.ReactNode`                     | -          |

## Accessibility

### Keyboard interactions

| Key     | Description                       |
| ------- | --------------------------------- |
| `Space` | Activates/deactivates the toggle. |
| `Enter` | Activates/deactivates the toggle. |
