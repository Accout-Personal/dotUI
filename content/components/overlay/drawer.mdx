## Installation

<Steps>
  <Step>Install the following dependencies:</Step>

```bash
npm install vaul
```

<Step>Copy and paste the following code into your project.</Step>
<ComponentSource name="components/core/default/drawer" />
<Step>Update the import paths to match your project setup.</Step>
<Step>Add the following data attribute to your root layout</Step>

To enable background scaling, it requires an element with [vaul-drawer-wrapper] data attribute to scale its background

```jsx
<body>
  <div vaul-drawer-wrapper>
    <App />
  </div>
</body>
```

</Steps>

## Usage

Only use a Drawer on small viewports. Shown here at any viewport for demonstration.

```jsx
import { Drawer } from "@/components/core/drawer";
```

```jsx
<Button onClick={()=> setOpen(true)}>Show</Button>
<Drawer open={open} onOpenChange={(show) => setOpen(!show)}>
  Button
</Drawer>
```

### Drawer

This component inherits and merges props from DrawerRoot and DrawerContent parts.
