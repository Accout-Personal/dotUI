
```jsx

```

## Best pratices

When possible, use a determinate progress indicator. An indeterminate progress indicator shows that a process is occurring, but it doesnâ€™t help people estimate how long a task will take. A determinate progress indicator can help people decide whether to do something else while waiting for the task to complete, restart the task at a different time, or abandon the task.
